Long-lasting sensors

Track, trace & sensor health of building components

Our sensors are developed with the aim to outlive buildings.
It is therefore designed to function in a mesh network using 6LoWPAN, Thread powered routing & UDP, with ZigBee on the application layer.

The sensors are encoded to be in sleep mode until changes occur, or when specified.
